# CV Application

A React-based CV Builder that allows users to create and edit their professional resume through an interactive form interface. Users can enter personal information, educational background, and work experience, then preview the formatted CV directly on the page.

## About The Project

This project was built as part of The Odin Project React curriculum.

The application demonstrates core React concepts including:

* Functional Components
* State Management with useState
* Props
* Controlled Components
* Conditional Rendering
* Event Handling
* Form Submission
* Component-Based Architecture

Users can:

1. Enter personal information.
2. Add educational qualifications.
3. Add practical work experience.
4. Submit each section individually.
5. Edit previously submitted information.
6. View a formatted CV preview.

---

## Features

### General Information

* Full Name
* Email Address
* Phone Number

### Educational Experience

* School Name
* Title of Study
* Date of Study

### Practical Experience

* Company Name
* Position Title
* Main Responsibilities
* Employment Start Date
* Employment End Date

### Edit Functionality

After submitting any section:

* Input fields are hidden.
* Information is displayed as formatted text.
* Edit button restores the form with previously entered values.

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/cv-application.git
```

Navigate to the project folder:

```bash
cd cv-application
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```
